## Comandos útiles

```bash
# Backend (desde backend/)
python -m uvicorn src.api.main:app --reload --host 0.0.0.0 --port 8000

# Frontend (desde frontend/)
npx vite

# Sincronizar correos vía API
curl -X POST http://localhost:8000/api/v1/emails/sync \
  -H "Authorization: Bearer $(TOKEN)"

# Sincronizar correos vía Python
python -c "
import httpx
r = httpx.post('http://localhost:8000/api/v1/auth/login',
    json={'username':'admin','password':'<ADMIN_PASSWORD_DEMO>'})
token = r.json()['access_token']
r2 = httpx.post('http://localhost:8000/api/v1/emails/sync',
    headers={'Authorization': f'Bearer {token}'})
print(r2.json())
"
```
